# Firefox Flatpak Repo
### For development purpose only
---

* Install
```
flatpak install --from https://raw.githubusercontent.com/cho2/firefox-flatpak-repo/master/firefox.flatpakref
```

* Run
```
flatpak run org.mozilla.Firefox
```
